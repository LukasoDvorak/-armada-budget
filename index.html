ARMADA BUDGET — USER GUIDE

OVERVIEW

Armada Budget is a single-page film production budgeting tool. All data is stored in your Dropbox — no server, no account required beyond Dropbox.

CONNECTING DROPBOX

Click ☁ Dropbox in the top bar and follow the authorisation flow. Once connected, your projects and rates are synced automatically. The last opened project loads on startup across all your devices.

NAVIGATION

Three tabs at the top switch between views:

  • Budget — the main line-by-line budget
  • Project Info — client, agency, director, deliverables, buyout
  • Summary — category totals, fee, insurance, grand total; PDF & Excel export

BUDGET — CATEGORIES

The budget is made up of categories (A, B, C…). Each category has one of four types:

  • Crew — prep days × rate + shoot days × rate. Supports overtime and transport sub-sections.
  • Transport — vehicles with sub-rows (day rate, mileage, overtime).
  • Std — simple qty × rate rows (post-production, studio, etc.).
  • Talent — casting sessions and talent with sub-rows and automatic agency fee calculation.

Click a category header to expand/collapse it. Drag rows by the ⠿ handle to reorder. Right-click a row to delete it. Double-click any label to rename it inline.

EDITING VALUES

Click any number cell to enter edit mode — the raw number input appears. Press Tab or Enter to confirm and move on. The display shows the formatted value with CZK when not editing. Qty columns (days, km, count) show plain numbers; Rate and Total columns show CZK.

FEE & INSURANCE

Production fee and insurance percentages are set in the settings panel (gear icon or Project Info tab). You can switch either to a flat amount. Fee is calculated on the subtotal; insurance on subtotal + fee.

Enable 💰 Fee mode to show a checkbox column — uncheck any row to exclude it from the fee base.

WORKING & ACTUAL MODES

These are two independent data layers that sit alongside your budget without changing it:

  • 🔧 Working — internal pre-production estimates. Green = under budget, red = over.
  • 📌 Actual — real costs as the shoot progresses. Same colour logic.

Right-click the Working or Actual button for options to clear or copy data between layers. W GRAND and A GRAND totals appear in the sticky bar at the bottom.

OVERCOST

Enable ⚠ Overcost to add a special section for costs that exceed the original budget. These roll up into their own total and are tracked separately in Working/Actual.

SAVING & VERSIONS

  • Autosave — runs automatically every 3 minutes and writes to autosave.json. Shown as a green ⚡ chip in the version bar.
  • 💾 Save — creates a timestamped checkpoint file in Dropbox. Use this for milestones.
  • Save As / New Version — available in the Save dropdown.

The version navigator (◀ ▶ chips below the project name) lets you browse and load previous checkpoints. Opening a project always loads the latest autosave if one exists.

ALTERNATIVE CURRENCY

Enable the alt currency toggle in the settings panel to show a second column with values converted at your chosen rate (e.g. EUR at 25 CZK). Appears in the budget, summary, PDF and Excel exports.

EXPORT

  • PDF — full formatted budget from the Summary tab.
  • Excel — two-sheet workbook (summary + detail) from the Summary tab.
  • Project Export / Import — share a full project JSON with a colleague via File menu.

SESSION LOCK

If you open the same project in two tabs or on two devices simultaneously, a warning appears. The lock expires after 1 minute of inactivity and is released automatically when you close the tab. Use 🔒 Lock to manually lock the UI against accidental edits.

INSTALLING AS AN APP (PWA)

Mac (Safari): File → Add to Dock

iPad / iPhone (Safari): Share → Add to Home Screen
